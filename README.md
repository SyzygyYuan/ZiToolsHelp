# 字统网非官方使用说明（稿）

字统网对自己的介绍是：汉字源、形、音、义、码网罗站点。简单地说，在字统网，你可以：
1. 源：查询汉字的字源
2. 形：观察字形的演变序列
3. 音：调查汉字韵书音、方言音和上古拟音
4. 义：
5. 码：调查汉字在不同字符集中对应的编码
此外还有很多其他功能。

# 检字方法
## 直接输入
页面右上方有一个文本框，可以直接在其中输入汉字或 Unicode 码，也可以输入一些部件，字统会提示一些备选项。

输入两个字，点击小喇叭按钮，可以进入对比两字读音界面。

输入两个以上的字，点击小喇叭按钮，可以同时对比多字读音。
